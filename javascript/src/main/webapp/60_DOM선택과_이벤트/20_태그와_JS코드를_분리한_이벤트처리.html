태그와 JS코드를 분리한 이벤트처리

            1.같은의미 , 다른표현
                태그 = 선택자 = 요소 = Element = DOM = Node

            2.자바스크립트 관점에서 태그(DOM)의 역할 

                JS가 HTML소스코드에 접근하여 내용이나 속성값
                또는 적용된 CSS코드를 조작하기 위한 통로(인터페이스)역할

            3. 실무에서 현재 가장 많이 사용되는 이벤트 구문 작성방식 
                DOM을선택.addEventListner('이벤트명',이벤트핸들러);; 
                DOM을선택.addEventListner('이벤트명',익명함수);;  
                DOM을선택.addEventListner('이벤트명',function(){});;
